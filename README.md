# UPI Transactions Analysis - Dashboard

## Overview

This Power BI dashboard provides a comprehensive and interactive analysis of UPI transaction patterns across multiple cities in India. It consolidates transactional data to help users understand payment behaviors, track transaction volumes, and monitor balance trends across different demographics and payment channels.​
The dashboard is designed for banking analysts, financial managers, and business intelligence teams who require real time insights into digital payment ecosystems.

## Key Features
### Transaction Performance Overview
  * Transaction Amount by Month with trend visualization for 2024
  * Total transaction amounts ranging from 1.6M to 1.7M per month
  * Remaining balance tracking across all transactions
  * Multi-currency support (INR, USD, EUR, GBP)
  * Interactive Filtering Capabilities
  * Bank Name filters (Sent and Received)
  * City-based analysis (Bangalore, Delhi, Hyderabad, Mumbai)
  * Device Type segmentation
  * Gender and Age Group demographics
  * Merchant Name filtering
  * Payment Method categorization
  * Transaction Purpose tracking
  * Transaction Type classification
  * Month-over-month trend analysis
  * Detailed transaction data with city, currency, month, amount, and remaining balance
  * Multi-dimensional slicers for granular data exploration

## Insights
* Transaction volumes remained relatively stable throughout 2024, with monthly amounts ranging between 1.60M and 1.71M.​
* March recorded the highest transaction amount at 1.71M, while August showed the lowest at 1.60M.​
* The dashboard supports multi-city analysis across major metros including Bangalore, Delhi, Hyderabad, and Mumbai.​
* Average remaining balances across cities ranged between 82M and 85M, indicating consistent liquidity patterns.​
* The platform provides comprehensive filtering across 10+ dimensions including demographics, geography, and transaction characteristics, enabling deep-dive analysis.​
